# TES - Terminal Exportador de Santos

TES - Terminal Exportador de Santos
Página institucional (HTML)

## Arquivos / Estrutura

Este projeto usa 3 arquivos, e os 3 precisam estar SALVOS NA MESMA PASTA:

<i class="fa-brands fa-html5 fa-2x"></i> Hyper Text Markup Language - index.html (contém toda a estrutura / conteúdo da página)
<i class="fa-brands fa-css3-alt fa-2x"></i> Cascading Style Sheets - styles.css (contém toda a aparência: cores, fontes, espaçamentos, animações etc.)
<i class="fa-brands fa-js fa-2x"></i> Java Script - script.js (contém todo o comportamento: menu mobile, scroll, formulário etc.)

## Abrir

Abra `index.html` no navegador ou publique a pasta inteira em qualquer hospedagem estática.

## Estrutura

- `index.html` — conteúdo, semântica, SEO básico e acessibilidade.
- `styles.css` — design system responsivo e animações que respeitam redução de movimento.
- `script.js` — menu mobile, revelação progressiva, botão de topo e resposta do formulário demonstrativo.
